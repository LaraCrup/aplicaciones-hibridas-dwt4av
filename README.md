- Configrar el usuario
```` bash
  git config --global user.name "Lara"
  git config --global user.email "laracrupni@gmail.com"
````

- Inicio del repositorio
```` bash
  git init
````

- Agrego los archivos
```` bash
  git add .
````

- Realizo el commit
```` bash
  git commit -m "Clase 03"
````
